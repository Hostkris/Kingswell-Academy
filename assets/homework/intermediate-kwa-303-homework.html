Big Idea

Language & Communication Growth is about learning how to grow language skill through speaking, listening, vocabulary, and repair. The student should finish the class able to explain the idea, show an example, correct mistakes, and save proof.

Self-Check Rubric

  • Concept understanding: Mastery — Explains Spanish greetings and Spanish sentence pattern clearly in plain language. Developing — Names the topic but cannot explain how it works.
  • Application: Mastery — Uses the skill correctly in holding a basic conversation or short presentation. Developing — Only repeats the classroom example or guesses.
  • Correction: Mastery — Finds one error, fixes it, and explains why the new answer is stronger. Developing — Changes the answer without explaining the reason.
  • Portfolio proof: Mastery — Submits a complete phrase bank + speaking practice log with reflection. Developing — Submits incomplete proof or a screenshot with no thinking shown.
