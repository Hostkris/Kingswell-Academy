Big Idea

Reading Comprehension & Study Skills is about learning how to turn reading into proof, not guesswork. The student should finish the class able to explain the idea, show an example, correct mistakes, and save proof.

Written Response

Practice Drills

  • Define claim in your own words and connect it to active reading.
  • Choose one example of main idea from reading one page of a lesson, article, or textbook and explain the evidence.
  • Find one common mistake students make with supporting details and correct it.
  • Create portfolio proof for Reading Comprehension & Study Skills.

Worked Examples to Review

  • Guided example: Active reading: Name the target skill: active reading.; Find where it appears in reading one page of a lesson, article, or textbook.; Use one vocabulary term such as claim or main idea to explain it.; Write one correction: what was unclear, wrong, missing, or unsupported?; Save the proof as a annotated page + five-sentence summary.
  • Transfer example: Annotation: Choose a fresh example involving annotation.; Complete the task once with guidance.; Mark one mistake or weak sentence/problem/decision.; Fix it and explain why the fix is better.; Teach the idea back in two or three plain sentences.

Project Proof

Build a personal study map for one course module

  • Pick one real situation connected to reading one page of a lesson, article, or textbook.
  • Use at least four course terms: claim, main idea, context clue, summary.
  • Create a first attempt, then revise it after feedback.
  • Submit the final annotated page + five-sentence summary and a short reflection.

Deliverable: Final deliverable: annotated page + five-sentence summary plus a one-paragraph reflection explaining what improved.

Self-Check Rubric

  • Concept understanding: Mastery - Explains active reading and main idea clearly in plain language.; Still developing - Names the topic but cannot explain how it works.
  • Application: Mastery - Uses the skill correctly in reading one page of a lesson, article, or textbook.; Still developing - Only repeats the classroom example or guesses.
  • Correction: Mastery - Finds one error, fixes it, and explains why the new answer is stronger.; Still developing - Changes the answer without explaining the reason.
  • Portfolio proof: Mastery - Submits a complete annotated page + five-sentence summary with reflection.; Still developing - Submits incomplete proof or a screenshot with no thinking shown.
