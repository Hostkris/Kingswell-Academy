Big Idea

Architecture Basics is about learning how to see buildings as decisions about space, people, and purpose. The student should finish the class able to explain the idea, show an example, correct mistakes, and save proof.

Self-Check Rubric

  • Concept understanding: Mastery — Explains design process and space clearly in plain language. Developing — Names the topic but cannot explain how it works.
  • Application: Mastery — Uses the skill correctly in studying a room, house, store, or campus layout. Developing — Only repeats the classroom example or guesses.
  • Correction: Mastery — Finds one error, fixes it, and explains why the new answer is stronger. Developing — Changes the answer without explaining the reason.
  • Portfolio proof: Mastery — Submits a complete labeled sketch + design notes with reflection. Developing — Submits incomplete proof or a screenshot with no thinking shown.
