# Kingswell Academy — Architecture + Drafting Curriculum Module

This is a GitHub Pages-ready HTML/CSS/JavaScript module for an Architecture + Drafting course.

## What is included

- Basic, Intermediate, and Advanced curriculum levels
- 12 classes per level
- Teacher-written lesson summaries
- Objectives
- Vocabulary
- Homework/studio assignments
- Rubrics
- Class quizzes
- 40-question level tests
- Automatic scoring
- Answer explanations
- Remediation messages
- Local progress tracking with `localStorage`
- Certificate unlock logic after passing a level test

## Files

```text
index.html
styles.css
data/architecture-drafting-curriculum.js
js/progress-tracker.js
js/quiz-engine.js
js/app.js
```

## How to use on GitHub Pages

1. Upload this folder to your GitHub repository.
2. Make sure `index.html` is at the root of the folder you are deploying.
3. Go to GitHub repository settings.
4. Open **Pages**.
5. Choose your branch and root folder.
6. Save and wait for GitHub Pages to publish.

## How the course is structured

The books are used as a source map only. The lesson content, questions, answer explanations, homework, rubrics, and scoring are original platform content.

Source map:

- Architectural Graphics — drawing language, linework, plans, sections, elevations, perspective, rendering, presentations
- Architecture: Form, Space, and Order — primary elements, form, space, organization, circulation, scale, ordering principles
- Building Construction Illustrated — site, foundations, floors, walls, roofs, moisture protection, thermal protection, doors/windows, MEP, materials
- Residential Design Using AutoCAD 2026 — CAD setup, floor plans, elevations, sections, site plans, schedules, plotting

## Important

This version saves progress only on the same browser/device. For real accounts, teacher dashboards, and cross-device grades, connect the same data model to a secure backend such as Firebase or Supabase.
