- Reading + Writing module in reading-writing/.

- Science + Biology + Chemistry module in science-biology-chemistry/.

Integration method:
- The root portal owns the main navigation, campus, original tests, homework library, tutor, sources, portfolio, and PWA shell.
- Each standalone curriculum app stays in its own folder and is mounted through an iframe from the root portal.
- This avoids JavaScript/global variable conflicts while keeping the uploaded files intact and working.
- Each module can also be opened directly through its own index.html.
