# Kingswell Academy Logic + Research Curriculum Platform

This is a GitHub Pages-ready client-side curriculum module for Logic + Research.

## Includes

- Basic, Intermediate, and Advanced course levels
- Curriculum data
- Lesson content
- Homework assignments
- 40-question final tests per level
- Lesson quizzes
- Auto-grading
- Answer explanations
- Skill scoring
- Book/source mapping
- Rubrics
- Local progress tracking

## How to use on GitHub Pages

1. Upload all files to your GitHub repository.
2. Go to Repository Settings.
3. Open Pages.
4. Choose Deploy from branch.
5. Select the main branch and root folder.
6. Save.
7. Open your GitHub Pages URL.

## Important

This version stores progress in the browser using localStorage. That is fine for a demo or personal learning platform.

For real student accounts, official grades, teacher dashboards, transcripts, or protected records, upgrade to a backend such as Firebase, Supabase, or your own server.
